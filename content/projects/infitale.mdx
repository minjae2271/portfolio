This is a AI children book generator built with NextJs and OpenAI API. It uses the
`@langchain/openai` package to handle connection to AI model and the `@ChatOpenAI`
library to set templates for messages sent to AI model and response from it.

## Features

- **Generating story**: Accept keywords and send request to OpenAI.
- **Choosing keywords**: Display keywords in a flex layout.
- **Image generating**: Add Image to each chapter only when the user visits chapter for the first time.
- **User Authentication**: Allow users to sign up and log in.
